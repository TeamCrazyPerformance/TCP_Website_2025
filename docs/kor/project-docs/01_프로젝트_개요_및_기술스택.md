## 프로젝트 개요

### TCP Website 2025란?

TCP Website 2025는 대학교 컴퓨터 과학 동아리를 관리하기 위한 풀스택 웹 애플리케이션입니다. 플랫폼이 제공하는 기능:

- **회원 관리**: 사용자 등록, 프로필, 인증 ([프로필 이미지 로직](../PROFILE_IMAGE_LOGIC.md) 참고)
- **스터디 그룹**: 학습 리소스 및 진행 상황 추적과 함께 스터디 그룹 생성 및 관리
- **팀/프로젝트 모집**: 프로젝트를 위한 팀 모집 게시 및 관리
- **공지사항**: 동아리 공지사항 및 뉴스
- **관리자 대시보드**: 동아리의 모든 측면을 관리하는 종합 관리자 패널
- **리크루팅 시스템**: 이력서 제출을 포함한 신규 회원 지원 처리

### 프로젝트 목표

- 동아리 활동을 위한 중앙 집중식 플랫폼 제공
- 회원 모집 및 온보딩 간소화
- 스터디 그룹과 프로젝트 팀을 통한 협업 촉진
- 효율적인 동아리 관리 실현

### 인프라

| 구성 요소 | 기술 | 용도 |
|-----------|------|------|
| **리버스 프록시** | Nginx 1.25-alpine | 웹/API 서비스로 트래픽 라우팅 |
| **정적 웹 서버** | Nginx 1.25-alpine | 프론트엔드 빌드 파일 서빙 |
| **데이터베이스** | PostgreSQL 16-alpine | 영구 데이터 저장소 |
| **로깅 스택** | ELK Stack 8.9.0 | 중앙 집중식 로깅 및 모니터링 |
| **컨테이너 오케스트레이션** | Docker Compose | 다중 컨테이너 관리 |

### 개발 도구

- **Prettier**: 코드 포맷팅
- **ESLint**: 코드 린팅
- **Jest**: 테스트 프레임워크
- **Supertest**: HTTP 테스트
- **SWC**: 빠른 Rust 기반 컴파일러 (NestJS 빌드 도구로 사용)
