## 🚀 빠른 시작

### 처음 시작하는 개발자라면?

1. **[프로젝트 개요](./01_프로젝트_개요_및_기술스택.md)**부터 읽고 전체적인 그림을 파악하세요
2. **[개발 가이드](./05_개발_및_배포_가이드.md)**를 따라 로컬 환경을 설정하세요
3. 관심 있는 영역(프론트엔드 또는 백엔드)의 아키텍처 문서를 읽으세요
4. 코드를 수정하기 전에 **[운영 및 문제 해결](./06_운영_및_문제해결.md)**을 참고하세요

### 배포 담당자라면?

1. **[인프라 및 배포](./02_인프라_및_배포.md)**에서 시스템 아키텍처를 이해하세요
2. **[개발 및 배포 가이드](./05_개발_및_배포_가이드.md)**의 배포 체크리스트를 확인하세요
3. **[CI/CD 도구](./05_개발_및_배포_가이드.md#cicd--운영-도구)** 섹션에서 자동화 스크립트를 학습하세요
