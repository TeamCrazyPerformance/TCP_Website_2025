## 운영진 모집 대상

TCP 웹서비스 운영팀은 **경력없는 신입, 경력있는 신입 모두를 환영해요**. 웹서비스 경험이 없던 사람은 실제 웹서비스가 어떻게 운영되는지를 경험할 수 있고, 경험이 있는 사람은 더 나아가서 웹서비스를 자신이 원하는 방향대로 발전시킬 수 있는 기회가 될 거예요.

**기존 TCP 부원이 아니어도 지원 가능해요!** (서비스 기획 & 운영 매니저 제외)

운영진으로 선발되시면 **TCP 정회원 자격이 자동으로 부여**돼요.

# 모집 역할

## 1. 🏗️ 서버 인프라 매니저 (Infrastructure Manager)
**"우리 서비스가 사는 튼튼한 집을 지어요"**

### 📝 이런 일을 해요
*   **서버 관리**: Proxmox라는 가상화 서버를 모니터링하고, 전원이 꺼지지 않도록 돌봐요. 특히 데이터가 유실되지 않도록 정기적인 백업 플랜을 세우고 관리해요.
*   **네트워크 설정**: 도메인(주소)을 연결하고, 외부에서 안전하게 접속할 수 있도록 문(포트)을 관리해요.
*   **하드웨어 관리**: 물리 서버가 있는 **TCP 인프라실**을 관리하고, 서버 부품(CPU, RAM, Disk)에 문제가 없는지 확인하고 점검해요.
*   **보안 담당**: 해커가 침입하지 못하도록 방화벽을 세우고, SSL 인증서(자물쇠)를 채워요.

### 🌟 이런 경험을 할 수 있어요
*   **내 손으로 만드는 진짜 서버**: AWS 같은 클라우드가 아니라, 실제 물리 서버를 처음부터 끝까지 만져볼 수 있어요.
*   **네트워크 전문가**: IP, 포트, DNS 같은 네트워크 핵심 개념을 책이 아닌 실전으로 배울 수 있어요.

### 🛠️ 이런 기술을 사용해요 (Tech Stack)
*   **OS/Server**: Proxmox VE, Ubuntu Linux
*   **Network**: IPTables, Nginx (Reverse Proxy), SSL/TLS
*   **Tools**: Shell Script (Bash), SSH

### 🙆 이런 분이면 더 좋아요
*   리눅스(Linux) 터미널 화면이 낯설지 않으신 분
*   "내 컴퓨터를 서버로 만들어보고 싶다"는 생각을 해보신 분

## 2. ⚙️ 백엔드 & DB 개발자 (Backend Developer)
**"눈에 보이지 않는 중요한 기능과 소중한 데이터를 책임져요"**

### 📝 이런 일을 해요
*   **API 개발 및 배포**: 사용자의 요청(로그인, 글쓰기 등)을 처리하는 로직을 만들고 서버에 올려요.
*   **데이터베이스 관리**: 회원 정보나 게시글 같은 데이터를 안전하게 저장하고 구조(Schema)를 설계해요.
*   **에러 해결**: "로그인이 안 돼요" 같은 문제가 생기면 로그를 보고 원인을 찾아 고쳐요.
*   **환경 설정**: 서버가 잘 돌아가도록 환경변수를 안전하게 관리해요.

### 🌟 이런 경험을 할 수 있어요
*   **대규모 트래픽 경험**: 실제 사용자가 쓰는 서비스의 트래픽을 처리하며 성능 최적화를 고민해볼 수 있어요.
*   **데이터 설계 역량**: 복잡한 데이터를 효율적으로 저장하고 불러오는 DB 설계 능력을 기를 수 있어요.
*   **CI/CD 파이프라인**: 코드를 짜면 자동으로 배포되는 자동화 시스템을 경험할 수 있어요.

### 🛠️ 이런 기술을 사용해요 (Tech Stack)
*   **Language**: TypeScript, Node.js
*   **Framework**: NestJS
*   **Database**: PostgreSQL, TypeORM
*   **Infra**: Docker, Docker Compose
*   **Auth**: JWT, Passport.js

### 🙆 이런 분이면 더 좋아요
*   NestJS, Node.js, SQL 중 하나라도 들어보셨거나 관심 있으신 분
*   논리적으로 문제를 해결하는 과정을 즐기시는 분
